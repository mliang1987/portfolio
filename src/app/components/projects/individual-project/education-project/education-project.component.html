High School Coding Curriculum: Game-Based Learning at Brooke High School

At Brooke High School, I developed a hands-on programming curriculum that introduced students to fundamental coding concepts through engaging, interactive projects. By incorporating game-based learning, students learned object-oriented programming, algorithmic problem-solving, and event-driven development in a dynamic way.

Curriculum Highlights

  • Interactive Coding – Students learned by modifying and expanding real-world projects.
  • Game Development Approach – Lessons focused on Java, Processing, and game logic, reinforcing core programming skills.
  • Problem-Solving Skills – The curriculum emphasized critical thinking, data structures, and software design principles.

Featured Starter Projects

  • Processing Demos – Introduced visual programming concepts, helping students manipulate graphics through Processing. View Project
  • Battleship Game Simulator – Reinforced data structures and game logic using Java-based mechanics. View Project
  • Starfighter Game – Focused on physics-based movement and interactive gameplay, encouraging creativity in game development. View Project

By working on these projects, students developed the confidence to modify, optimize, and create their own software applications, preparing them for future opportunities in computer science and software engineering. My goal was to bridge the gap between theory and practice, fostering innovation and curiosity through real-world coding applications.

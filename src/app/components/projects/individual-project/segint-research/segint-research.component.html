SegInt-Research is an on-premise auto-segmentation transaction server designed for research institutions working with machine learning models. It integrates with Velocity, Varian's bioinformatics solution, to consolidate imaging data from various sources. The server operates locally, allowing research teams to develop and test their own segmentation models without relying on cloud-based services.

The architecture of SegInt-Research includes:

  • Communication via Microsoft Radiomics Segmentation API (Project InnerEye).
  • Message serialization using Google's protocol buffers.
  • Django-based endpoints with SQLite as a shared data layer.
  • Asynchronous segmentation jobs managed by Celery, with Redis as a message broker.

SegInt-Research is Linux-native, developed on Ubuntu 18.04 LTS, and can be installed manually or virtualized using Hyper-V or VirtualBox. It requires network access, at least 15GB of disk space, and 2GB of memory to function properly.

SegInt-Research was a developed as a project solely by me for my internship at Varian. It is also now used internally by the Varian testing team to test the Velocity software against a SegInt-compliant service provider. More detailed information can be found in the GitHub readme below.
